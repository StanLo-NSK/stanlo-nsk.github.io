= Сервис работы с задачами

Настоящий документ содержит техническое описание HTTP-сервиса работы с задачами, структура данных которых приведена <<Структура данных сервиса работы с задачами, ниже>>.

Сервис позволяет:

. Создавать задачи (xref:post/post.adoc[POST]).
. Получать список созданных задач (xref:get-all/get-all.adoc[GET]).
. Получать конкретную задачу из списка (xref:get/get.adoc[GET]).
. Изменять параметры задачи (PUT).
. Удалять задачу (DELETE).
